# UltraBugs

_UltraBugs_ is a tactical turn-based game built for the [Game Off 2021](https://itch.io/jam/game-off-2021) game jam using [LÖVE](https://love2d.org/).

## Requirements

- LÖVE v11.3
- Aseprite v1.2

## Development

To run the game do:

```sh
$ make run
```
